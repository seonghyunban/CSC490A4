\subsection{Codebase Organization for Part 4}
\nestindent

This project spans two repositories: the \texttt{nanochat} training framework, which was extended with a modular reward system, and the \texttt{AI-Accountant} experiment repository, which houses the pipeline scripts, run configurations, and collected results for the 10-run ablation study.
A snapshot of the modified nanochat files is kept under \texttt{a4/p4/nanochat/} so that the experiment directory is self-contained and reproducible without depending on a specific nanochat commit.
