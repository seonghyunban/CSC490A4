The expanded reward set (C, D, E) increases the ablation from six to ten runs.
In addition to the original baseline, separate A/B, and combined A+B, we add: separate C, separate D, separate E, combined C+D, combined C+D+E, and combined all (A+B+C+D+E).
The combined C+D run tests the dependency hypothesis that coherence (C) is a prerequisite for entity grounding (D) to be effective.
The combined all run tests whether all five rewards interact constructively.
