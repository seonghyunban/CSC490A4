The expanded reward set (C, D, E) increases the ablation from four to ten runs.
In addition to the original Baseline, + A, + B, and A+B, we add: + C, + D, + E, C+D, C+D+E, and Comb.\ All (A+B+C+D+E).
The C+D run tests the dependency hypothesis that non-degeneracy is a prerequisite for entity grounding to be effective.
The Comb.\ All run tests whether all five rewards interact constructively.
