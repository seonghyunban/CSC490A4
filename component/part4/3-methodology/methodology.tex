\subsection{Methodology}

We adopt an ablation-style experimental design: all runs share the same SFT starting checkpoint and identical RL hyperparameters, varying only the reward function composition.
This isolates the effect of each reward addition.
We observe two measurement points per run: training-time metrics logged to W\&B every step, and a full post-training evaluation on the GSM8K test set (1319 problems, 8 samples each).
